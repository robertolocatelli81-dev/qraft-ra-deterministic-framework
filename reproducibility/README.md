# Riproducibilità — QRAFT-RA (V13 + V14)

Questa cartella contiene gli artefatti di riproduzione deterministica del framework QRAFT-RA,
associati al record Zenodo:

DOI: https://doi.org/10.5281/zenodo.18462332

Lo scopo è consentire a qualsiasi revisore o lettore di riprodurre
in modo indipendente i risultati numerici certificati riportati nel paper,
senza alcun campionamento Monte Carlo.
